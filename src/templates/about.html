Features

Main Features

  • ✔️ Lightweight, JS-free design
  • ✔️ Multiple CV templates
  • ✔️ Choose between PDF or DOCX output
  • ✔️ Simple, clean UI/UX inspired by 2000s design
  • ✔️ Fully responsive, mobile-friendly layout
